TRANSPARENCIA




Codigo Descripción Ejemplo
.img2:hover{
filter: opacity(20%) drop-shadow(8px 8px blue);
} 		Resultado:

Cuando el usuario pasa el mouse sobre el elemento con la clase .img2 (en este caso, un div con fondo amarillo), ocurrirán dos cosas:

La opacidad del elemento se reducirá al 20%, haciendo que se vuelva ligeramente transparente.
Se añadirá una sombra azul con un desplazamiento de 8 píxeles en ambas direcciones (horizontal y vertical).

Este efecto es útil para proporcionar realimentación visual cuando el usuario interactúa con elementos en la página web, haciendo que la experiencia de usuario sea más dinámica y atractiva.
.img3:hover{
filter: saturate(80%) drop-shadow(8px 8px blue);
} 		Resultado:

Cuando el usuario pasa el mouse sobre el elemento con la clase .img3 (en este caso, un div con fondo amarillo), ocurrirán dos cosas:

La saturación del elemento se aumentará al 80%, haciendo que los colores sean más intensos y vibrantes.
Se añadirá una sombra azul con un desplazamiento de 8 píxeles en ambas direcciones (horizontal y vertical).

Este efecto es útil para destacar visualmente elementos en respuesta a la interacción del usuario, mejorando la experiencia visual y la interactividad en la página web.
.img4:hover{
filter: opacity(80%) box-shadow(8px 8px blue);
} 		Resultado:

Cuando el usuario pasa el mouse sobre el elemento con la clase .img4 (en este caso, un div con fondo amarillo), ocurrirán dos cosas:

La opacidad del elemento se reducirá al 80%, haciendo que se vuelva ligeramente transparente.
Se añadirá una sombra azul alrededor del borde del elemento con un desplazamiento de 8 píxeles en ambas direcciones (horizontal y vertical).

Este efecto combinado de opacidad reducida y sombra añadida ayuda a resaltar visualmente el elemento cuando el usuario interactúa con él, proporcionando una experiencia de usuario más dinámica y atractiva en la página web.
